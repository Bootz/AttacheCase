ヘルプファイルについて
===================================

ヘルプファイルの作成には、  

HelpNDoc 3  
http://www.helpndoc.com/  
を使用しています。  

同ディレクトリにある「AttacheCase.hnd」がそのデータファイルになります。ひとつのバイナリデータにまとめられているため、上記のアプリケーションでないと開けません。  

このアプリケーションの大きな特徴として、Compiled HTML Help（いわゆるCHMファイル）への出力や、同時にWebサイト上に公開できる状態のHTMLファイルなどへマルチ出力できるという簡便さから導入しました。  

また、各ページにバナーが出てしまうものの、フリーでの使用も可、ということも採用理由となりました。  

以前はHTMLをテキストエディタで編集して、バッチファイルから、Microsoft HTML Help Workshopに直接コンパイルをかけていました。  

また、別に自家製ツールを作って、自身のサイト上に公開するページ出力するようにするなど、何かとプロセスがごちゃごちゃしていました。  

HelpNDocを使うと、pdfなども含めマルチ出力できるのはもちろん、挿入画像ファイルも含め、１ファイルで管理可能となります。今後はこちらで更新していきたいと思います。  

もし上記のヘルプファイルの多言語への翻訳を行っていただける方がおりましたら、僕のWebサイトからご一報いただけるとうれしいです。そのときは、ついでにアプリケーション内の文字列の翻訳も併せてお願いするかもしれませんが、よろしくお願いいたします。
