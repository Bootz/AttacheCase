ヘルプファイルの作成について
===================================

以前までのβバージョンでは、

HelpNDoc 3  
http://www.helpndoc.com/  

を使用してヘルプファイルの作成を行っていましたが、サイト上で使い回すこ
とを考えたり、また画像やデザインなどの一括管理には、プレーンなHTMLの方
が取り回しが良いため、けっきょく元に戻しました（すみません・・・）。


オンラインヘルプ
===================================

本体となるHTML群は、「pages」フォルダーに入っています。
サイト上では、

css
js
json
pages
index.html

をアップロードする予定でいます。

ちなみに「json」フォルダー内にあるHTMLは、サイト、ヘルプファイルで併用
する目的で、内容などをjsonファイルにしてあるため、ヘルプファイルに必要
なファイルに変換するツールをJavaScriptで書いてます。

アプリケーション付属のヘルプファイル
====================================

アプリケーションに付属させるヘルプファイルは、
Microsoftから無償で提供されている、

HTML Help Workshop
http://www.microsoft.com/en-us/download/details.aspx?id=21138

のインストールが必要となります。

インストール後に「help_file」フォルダー
にある、AttacheCase.hhp を「HTML Help Workshop」で開いて、
コンパイルすることで、最終的なヘルプファイル（AttacheCase.chm）を
出力することができます。
